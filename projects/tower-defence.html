Cosmic Frontline (Prototype)

A small sci-fi tower defence prototype built to experiment with A* pathfinding, enemy routing, and grid-based tower placement in Unity.

Overview

Cosmic Frontline is a technical prototype focused on the core systems behind a tower defence game rather than full content. Enemies spawn and move along a path toward the player’s base, while the player places towers on a grid to try and stop them. The main goal of the project was to learn and implement A* pathfinding for enemy navigation and dynamic path updates.

Details

  • Engine: Unity
  • Language: C#
  • Role: Solo developer (prototype design & programming)
  • Focus Areas: Implementing the A* pathfinding algorithm for enemy navigation, grid-based tower placement, basic turret attack logic, wave spawning system, and testing how paths update when the player changes the layout.
